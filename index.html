Welcome to my website!

I'm a PhD candidate working in the Functional Programming Lab at the University of Nottingham, UK. I'm interested in strongly typed functional programming, dependent type theory, interactive theorem proving, programming language theory and implementation, formalization of mathematics, program correctness, static analysis. I write most of my code in Agda and Haskell.

Projects

  • iit-encoding: Agda formalization of a method to reduce linear infinitary inductive-inductive types to mutual inductive families, without relying on function extensionality.
  • setoid-universe: Agda formalization of a universe of setoids in the setoid model of intensional type theory.
  • mokuro-tui, mokuro-reader: a terminal user interface (TUI) and web interface to produce and consume Japanese manga OCR data using mokuro. Written in Haskell and Javascript.
  • nbe-wes, nbe-stlc: Agda formalizations of proofs of Normalization by Evaluation for lambda calculi with weak conversion.
  • Zsyntax: ...
  • Jandom: static analyser based on abstract interpretation, written in Scala.
  • flight-routes: Big Data analysis of raw ADS-B data. Written in Apache Spark, runs on Hadoop clusters.
  • muKanren: ...
  • onyx-wm: minimal tiling window manager for Mac OS, written in Haskell and Objective-C through FFI. Most likely broken on new Mac OS versions.
